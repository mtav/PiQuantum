Detailed Description

Interrupts in Linux.

This class implements the SIGALARM function in Linux. SIGALARM can be setup to call a function at regular intervals. The function (interrupt) does not have an implementation in this class. It is defined in the derived InputOutput class. The constructor sets up the SIGALARM (defining the period, etc.)
